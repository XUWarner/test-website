# 制造业数字化是什么

利用数字技术提升制造业能力，对制造业研发生产全流程和产业链供应链各环节进行改造升级和价值重塑的过程，是制造业高质量发展的关键路径

- [x]  制造业数字化转型行动方案[关于工信部等三部门联合印发《制造业企业数字化转型实施指南》](https://www.wnd.gov.cn/doc/2024/12/27/4485359.shtml)

根据制造业多样化个性化需求，分行业分领域挖掘经典场景，做好设备联网、协议互认、标准制定、平台建设等工作

# 学习大纲

1. 制造业数字化的核心是什么
    1. 使用数字技术将生产设施联网操作
    2. 采集并整合数据，用于生产设施生命周期的模型建设，预测最佳维修时间点以提高收益（数字孪生）
    3. 保持单一信息源，便于内部环节更好磨合，优化整体流程结构
    4. 让工艺节拍保持一致，避免能源浪费或资源浪费的情况
2. 为什么制造业需要 AI，而不是技术堆栈
    1. 技术堆栈是应用软件的堆叠，仍然需要人的经验进行判断和总结应用，不能达到效率最大化
    2. 且人的精力有限，在生产制造设施达到一定的规模的时候难以调度，需要ai汇总合并，优化流程，保证效益
    3. 也有利于单一信息源的建立
3. 工业 AI 在几个关键场景的商业价值
    1. 视觉检测，通过24/7持续运作及成熟模型较高的识别率降低人工成本，提升检测效率
    2. 预测维护，通过引入时序模型对设备生命周期和故障时间点进行准确预测，降低应急关停设备造成的损失，最大程度发挥机器的生产价值
    3. 产线优化，通过AI高算力的特征组合多个生产参数进行实况模拟，在不同的组合中寻找全局最优解，达到单位时间最大产出，减少能耗，降本增效的体现
    4. 智能排产，通过AI运算，综合考虑交期、成本等因素，决定最优生产顺序，提高产线柔性，减少订单延期
4. 产品设计如何从“用户痛点”出发
    1. 一线调研分析，保持敏锐，用始终保持可优化改进的态度看环节
    2. 把情绪描述转换为清晰理性的问题描述
    3. 将问题难题拆解为多层次分布推出解决方案
    4. 各解决方案同步推动设计产品
5. 竞品与行业机会图谱
6. 制造业数字化的落地阻碍与解决路径
7. 针对辎云科技的调研简版（产品/客户/案例/价值）
